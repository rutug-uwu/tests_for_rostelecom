Позитивный сценарий: Получение подсказки за просмотр рекламы
Предусловия:
1)Открыт эмулятор Android 11 (API 30)
2)Устройство подключено к интернету
3)Установлено приложение "Алхимия:Головоломка"
4)Открыто приложение "Алхимия:Головоломка" и пользователь находится в блоке подсказок
Шаги:
1)Нажать "Watch" в блоке "2 hints - For watching ads" и дождаться завершения просмотра рекламы
2)Закрыть рекламу/вернуться в приложение
Ожидаемый результат:
Количество подсказок увеличилось на 2 единицы
