1. Позитивный сценарий: Видео должно воспроизводиться при наличии интернета
Предусловия:
1)Открыт эмулятор Android 11 (API 30)
2)Устройство подключено к интернету
3)Установлено приложение "VK Video"
4)Приложение VK Video открыто, отображается экран Main со списком видео
Шаги:
1)Нажать на первое видео в списке
Ожидаемый результат:
Открывается экран плеера, воспроизведение видео начато

2. Негативный сценарий: Видео не воспроизводится без интернета
Предусловия:
1)Открыт эмулятор Android 11 (API 30)
2)Устройство не подключено к интернету
3)Установлено приложение "VK Video"
Шаги:
1)Открыть вк видео
Ожидаемый результат:
Открывается экран Profile в офлайн режиме. На экране присутствует следующий текст:
Offline mode и No internet
